# 🎭 AI & Random Avatar Generator

Bu proje, kullanıcıların tamamen benzersiz, modern ve eğlenceli avatarlar üretmesini ve bunları bilgisayarlarına indirebilmesini sağlayan bağımsız bir **Vanilla JavaScript** web uygulamasıdır. Herhangi bir backend bağımlılığı veya API anahtarı gerektirmeden, doğrudan tarayıcı mimarisiyle çalışır.

## 🚀 Özellikler

* **Dinamik Tarz Seçimi:** Robotlar, piksel sanatı, insan figürleri ve minimalist geometrik çizimler dahil 5 farklı avatar kategorisi.
* **Benzersiz Algoritma:** `Math.random()` ve zaman damgası (`Date.now()`) kombinasyonu ile her tıklamada tamamen eşsiz ve tekrarlanmayan karakter tasarımları.
* **SVG İndirme Desteği:** Beğenilen avatarları şeffaf arka planlı yüksek kaliteli `.svg` formatında tek tıkla bilgisayara indirme imkanı.
* **Modern UI/UX:** CSS gradient geçişleri, siber ızgara (pixel grid) deseni ve cam efekti (glassmorphism) barındıran responsive tasarım.

## 🛠️ Kullanılan Teknolojiler

* **HTML5:** Semantik web yapısı.
* **CSS3:** Özel arka plan algoritmaları, animasyonlar ve modern kart tasarımları.
* **Vanilla JavaScript:** DOM manipülasyonu, sanal indirme köprüleri ve dinamik dökümantasyon entegrasyonu.
* **DiceBear API:** Açık kaynaklı vektörel avatar kütüphanesi.
